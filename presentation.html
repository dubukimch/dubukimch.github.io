DIGITAL TWIN / SMART FARM

스마트팜 통합 솔루션

여러 장비와 프로그램으로 분산된 스마트팜 운영을 하나의 디지털 트윈 기반 경험으로 통합하는 프로젝트

이의락 Blazor MQTT Unity WebGL

01. PROBLEM

왜 통합 솔루션이 필요했나

  • 새로운 장비가 기존 장비와 독립적으로 개발되어 호환이 어려움
  • 장비마다 별도 제어 소프트웨어를 익혀야 해서 운영 피로도가 높음
  • 2D 중심 UI와 전문 용어 중심 사용 방식 때문에 직관성이 떨어짐

02. SOLUTION

디지털 트윈 기반 통합 제어

모든 스마트팜 장비를 하나의 프로그램에서 전장 제어할 수 있도록 구성한 디지털 트윈 소프트웨어입니다.

통합 운영 여러 도구를 오가는 대신 단일 흐름으로 제어
시각적 이해 게임처럼 직관적인 3D UI 경험 제공
확장 가능 구조 새로운 장치와 기술 추가에 유리한 계층 설계

03. IMPACT

기대 효과

업무 간소화 분산된 프로그램을 하나로 줄여 관리 효율 향상
신기술 확장 표준화된 인터페이스로 새로운 기술 연결 기회 확보
기술 장벽 완화 복잡한 시스템을 친숙한 화면으로 전달

04. ARCHITECTURE

4계층 소프트웨어 구조

Arduino센서 제어 / 와이파이
Raspberry Pi입출력 제어 / 프록시
Linux Server데이터 저장 / 분석
ClientWebAssembly / 다중 OS

하드웨어는 비닐하우스, LED 스트립, 온습도 센서, 공기질 센서, 아크릴 파이프를 포함한 구조로 구성됩니다.

05. PROCESS

진행 과정

  1. 하드웨어 제작
  2. Arduino와 Raspberry Pi 연결
  3. Raspberry Pi와 Client 연결
  4. Blazor UI 개선
  5. WebGL 적용

06. SAMPLE CODE

샘플 코드 구성

  • Arduino LEDSensor - MQTT 기반 센서/LED 제어 구조
  • Raspberry Pi DeviceController - Ping 기반 장비 탐색과 계층 정보 제공
  • Main Server Devices - Tree UI와 연결 흐름
  • Unity WebGL Interop - Blazor와의 상호작용 처리
