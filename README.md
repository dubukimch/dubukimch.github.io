# 이의락 포트폴리오 URL용 정적 웹사이트

GitHub Pages, Netlify, Vercel 등에 바로 배포할 수 있는 HTML/CSS/JS 정적 포트폴리오입니다.

## 구성

- `index.html` : 메인 포트폴리오 페이지
- `assets/css/styles.css` : 전체 디자인 CSS
- `assets/js/main.js` : 모바일 메뉴, 섹션 하이라이트, 등장 애니메이션
- `assets/images/` : 포트폴리오 이미지
- `assets/docs/smartfarm-digital-twin-portfolio.pdf` : 원본 스마트팜 포트폴리오 PDF

## 배포 방법

1. ZIP 압축을 풉니다.
2. GitHub Pages 저장소 루트에 전체 파일을 업로드합니다.
3. `index.html`이 루트에 있어야 기본 URL로 접속됩니다.
4. 배포 전 `mailto:ddf****@naver.com`을 공개 가능한 실제 이메일 주소로 교체하세요.

## 개인정보 처리

공개 URL용으로 휴대폰 번호와 주소는 제외했습니다.
